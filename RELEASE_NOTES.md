# RGR06 Mapper v0.19

## 发布内容

- 将乐奇眼镜配套 RGR06 戒指映射为 Windows 电脑端鼠标和快捷键控制器。
- 默认支持鼠标移动、左键、右键、中键、滚轮和返回/后退等电脑操作。
- 设置窗口支持为每个动作选择功能，并通过“录制”绑定自定义按键。
- 程序常驻系统托盘，支持暂停/恢复映射和随 Windows 启动。
- 安装包内置 AutoHotInterception 依赖和 Interception 驱动安装器。

## 安装

1. 下载 `RGR06-Mapper-Setup-v0.19.exe`。
2. 以管理员权限运行安装包。
3. 安装完成后重启电脑。

## 注意

- 本程序面向 Windows 10/11 x64。
- Interception 是键盘/鼠标过滤驱动，安装和卸载后都建议重启电脑。
- 如果安全软件拦截驱动安装，请按目标电脑的安全策略处理后再安装。
